Cropme

Cropme is a customizable and easy to use javascript image cropping plugin

Cropme result

Installation

npm: npm install cropme

download source.zip

<link rel="stylesheet" href="croppie.css" />
<script src="croppie.js"></script>

Usage

vanilla javascript example

var el = document.getElementById('cropme')
var cropme = new Cropme(el, options)
cropme.method(options)

jquery example

var el = $('#cropme')
el.cropme(el, options)
el.cropme('method',options)
